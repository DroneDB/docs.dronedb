.. _pull_command:

********************************************************************************
pull
********************************************************************************

::

    Pulls changes from a remote repository.
    
    Usage:
      ddb pull
    
      -r, --remote arg  The remote Registry (default: )
      -f, --force       Forces the operation
      -h, --help        Print help
          --debug       Show debug output
